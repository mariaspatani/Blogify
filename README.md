# 📝 Blogify

**Blogify** is a modern blogging platform built with **Next.js**, **TypeScript**, **Tailwind CSS**, **Firebase Authentication**, **Firestore**, and **MockAPI**. It provides a clean, responsive interface for discovering, reading, and interacting with blog posts.

## ✨ Features

* 📖 Browse and read blogs
* 🔍 Search and filter articles
* 🔐 Secure Email & Password authentication
* 👤 User profile management
* ❤️ Like blog posts
* 💬 Comment on articles
* 🔖 Bookmark favorite blogs
* ✍️ Create and manage blogs (authenticated users)
* ⚡ Server-Side Rendering (SSR)
* 📱 Responsive design
* 🌙 Modern and intuitive UI

## 🛠️ Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Firebase Authentication
* Cloud Firestore
* TanStack React Query
* MockAPI
